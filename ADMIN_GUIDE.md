# 관리자 페이지 사용 가이드

## 접속 방법

1. **URL 접속**
   - `https://your-site.vercel.app/admin` 또는 `https://your-site.vercel.app/admin.html`

2. **PIN 입력**
   - 기본 PIN: `2222`
   - PIN은 `shared.js` 파일에서 변경 가능

## 수정 가능한 항목

### 1. 매장명
- 손님 페이지 상단에 표시되는 매장 이름

### 2. 필수 키워드
- **위치**: "필수 키워드" 섹션
- **형식**: 쉼표로 구분
- **예시**: `어국수, 국물, 시원`
- **설명**: 리뷰에 항상 포함되는 키워드

### 3. 프로모션 키워드 풀
- **위치**: "프로모션 키워드 풀" 섹션
- **형식**: 쉼표로 구분
- **예시**: `깔끔, 매콤, 어묵국수, 진한맛, 깊은맛`
- **설명**: 랜덤으로 2~4개 선택되어 리뷰에 포함됨

### 4. 메뉴 리스트
- **위치**: "메뉴 리스트" 섹션
- **형식**: 줄바꿈 또는 쉼표로 구분
- **예시**:
  ```
  어묵국수
  얼큰바지락국수
  소갈비국수
  잔치국수
  ```
  또는
  `어묵국수, 얼큰바지락국수, 소갈비국수, 잔치국수`
- **설명**: 손님이 선택할 수 있는 메뉴 목록

### 5. 사이드 메뉴 리스트
- **위치**: "사이드 메뉴 리스트" 섹션
- **형식**: 쉼표로 구분
- **예시**: `소주, 맥주, 김치전, 부추전, 없음`
- **설명**: "함께 먹은 것" 옵션 목록

### 6. UI 설정
- **배경색**: 페이지 배경색
- **강조색**: 버튼, 강조 요소 색상
- **부제목**: 손님 페이지 상단 부제목

## 저장 방법

### 방법 1: 저장 (로컬)
- **용도**: 같은 브라우저/기기에서만 유지
- **저장 위치**: 브라우저 localStorage
- **장점**: 즉시 반영
- **단점**: 다른 기기/브라우저에서는 적용 안 됨

### 방법 2: 설정 JSON 다운로드
- **용도**: 원격 서버에 업로드하여 모든 사용자에게 적용
- **다운로드**: `config.json` 파일 생성
- **업로드**: 다운로드한 파일을 서버의 루트에 업로드
- **장점**: 모든 사용자에게 동일한 설정 적용
- **단점**: 서버 파일 업로드 필요

### 방법 3: 설정 초기화
- **용도**: 기본값으로 되돌리기
- **주의**: 저장 버튼을 눌러야 적용됨

## 사용 팁

1. **키워드 작성 시 주의사항**
   - 비문을 유발하는 키워드는 피하세요 (예: "국물했어요", "어국수했어요")
   - 자연스러운 단어로 작성하세요 (예: "국물", "어국수", "시원")

2. **메뉴 추가/삭제**
   - 줄바꿈으로 한 줄씩 작성하면 보기 편합니다
   - 쉼표로 구분해도 됩니다

3. **저장 후 확인**
   - 메인 페이지(`/`)를 새로고침하여 변경사항 확인
   - 변경이 안 보이면 브라우저 캐시를 삭제해보세요

## PIN 변경 방법

`shared.js` 파일을 열어서 다음 줄을 수정:

```javascript
const ADMIN_PIN = '2222'; // 여기 숫자 변경
```

변경 후 Vercel에 재배포 필요합니다.

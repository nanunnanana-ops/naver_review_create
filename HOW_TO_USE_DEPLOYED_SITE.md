# 배포된 사이트 사용 방법

## 배포 완료 후

### 1. 배포된 URL 확인

Vercel 배포가 완료되면:
- ✅ "Congratulations!" 메시지 표시
- ✅ 배포된 URL이 표시됨
- 예: `https://naver-review-generator.vercel.app`

### 2. 사이트 접속 방법

#### 메인 페이지 (손님용)
```
https://배포된URL/index.html
```
또는
```
https://배포된URL/
```
(자동으로 index.html로 리다이렉트됨)

**예시:**
- `https://naver-review-generator.vercel.app/index.html`
- `https://naver-review-generator.vercel.app/`

#### 관리자 페이지
```
https://배포된URL/admin.html
```

**예시:**
- `https://naver-review-generator.vercel.app/admin.html`
- PIN: `2222`

### 3. 사이트 사용 방법

#### 메인 페이지 사용 (리뷰 생성)

1. **메인 페이지 접속**
   - 배포된 URL 접속
   - 예: `https://naver-review-generator.vercel.app/`

2. **메뉴 선택**
   - "메뉴 선택 (복수 선택 가능)" 섹션
   - 원하는 메뉴 체크박스 선택 (여러 개 선택 가능)
   - 예: 어묵국수, 얼큰바지락국수

3. **함께 먹은 것 선택** (선택사항)
   - "함께 먹은 것" 섹션
   - 원하는 항목 선택 (여러 개 선택 가능)
   - 예: 소주, 맥주

4. **리뷰 생성**
   - "리뷰 3개 만들기" 버튼 클릭
   - 잠시 대기 (Gemini AI가 리뷰 생성 중)
   - 약 5-10초 소요

5. **리뷰 확인 및 복사**
   - 생성된 리뷰 3개 확인
   - 원하는 리뷰의 "복사" 버튼 클릭
   - 자동으로 네이버 플레이스 페이지로 이동
   - 복사한 리뷰를 붙여넣기

#### 관리자 페이지 사용 (설정 변경)

1. **관리자 페이지 접속**
   - `https://배포된URL/admin.html` 접속

2. **PIN 입력**
   - PIN: `2222` 입력
   - "확인" 버튼 클릭

3. **설정 변경**
   - 매장명 변경
   - 필수 키워드 편집
   - 프로모션 키워드 풀 편집
   - 메뉴 리스트 편집
   - 사이드 메뉴 리스트 편집
   - UI 설정 (색상, 부제목)

4. **저장**
   - "저장 (로컬)" 버튼: localStorage에 저장 (같은 기기에서만 유지)
   - "설정 JSON 다운로드": config.json 파일 다운로드 (서버에 업로드 가능)

## 문제 해결

### 사이트가 안 열릴 때

1. **URL 확인**
   - 정확한 URL인지 확인
   - `https://`로 시작하는지 확인

2. **404 에러**
   - `/index.html`을 명시적으로 입력
   - 예: `https://배포된URL/index.html`

3. **배포 상태 확인**
   - Vercel 대시보드에서 배포 상태 확인
   - "Ready" 상태인지 확인

### 리뷰가 생성되지 않을 때

1. **환경 변수 확인**
   - Vercel Settings → Environment Variables
   - `GEMINI_API_KEY`가 설정되어 있는지 확인

2. **재배포**
   - 환경 변수를 추가했다면 재배포 필요
   - Deployments → 최신 배포 → Redeploy

3. **브라우저 콘솔 확인**
   - F12 → Console 탭
   - 에러 메시지 확인

### 복사 기능이 안 될 때

1. **HTTPS 확인**
   - Clipboard API는 HTTPS에서만 작동
   - Vercel은 자동으로 HTTPS 제공

2. **브라우저 권한 확인**
   - 브라우저가 클립보드 접근을 허용하는지 확인

## 공유하기

배포가 완료되면:

1. **URL 공유**
   - 배포된 URL을 친구들에게 공유
   - 예: `https://naver-review-generator.vercel.app`

2. **QR 코드 생성** (선택사항)
   - QR 코드 생성 사이트 사용
   - 모바일에서 쉽게 접속 가능

3. **북마크 추가**
   - 자주 사용할 사이트라면 북마크 추가
